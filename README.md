# Automounter

AI-монтажёр видео: анализирует дроновые съёмки, понимает запрос на естественном языке и автоматически собирает музыкальный клип с синхронизацией по ритму.

---

## Требования

- **Bun** (обязательно для OpenTUI FFI):
  ```bash
  curl -fsSL https://bun.sh/install | bash
  ```
- **Ollama** с установленными моделями:
  ```bash
  ollama pull llava:13b           # vision-анализ кадров при индексации
  ollama pull qwen2.5-coder:14b   # анализ текстового промпта
  ollama serve
  ```
- macOS с Apple Silicon рекомендуется (аппаратное кодирование через `h264_videotoolbox`). Linux/Windows использует `libx264`.

## Установка

```bash
bun install
```

## Использование

### Интерактивный режим (OpenTUI)

```bash
bun start
```

Открывается TUI-интерфейс:
- **Создать ролик** — новый проект: промпт, LUT, цветовой референс, качество, ориентация.
- **Правка последнего ролика** — быстрый ре-рендер с изменёнными параметрами.
- **Переиндексация папки** — обновить базу видео (GPS, теги, геокодирование).

### Пакетный / скриптовый режим

```bash
bun start -- --batch
```

### Индексация без монтажа

```bash
# Индексировать новые файлы (глубокий анализ + геоданные)
bun start -- --index-only --input /path/to/vid

# Полная переиндексация (очистка кэша)
bun start -- --reindex --input /path/to/vid
```

### Проверка типов

```bash
npx tsc --noEmit
```

## Логика ранжирования клипов

Клипы ранжируются по темам пользователя, а не по плоскому совпадению тегов.

**Пример промпта:** "природа с машиной"

| Клип | Темы | Оценка |
|------|------|--------|
| природа + машина | 2 темы (машина=required) | +12 + 24 = +36 |
| природа × 10 тегов | 1 тема | +12 |
| машина × 1 тег | 1 тема (required) | +24 |
| нет совпадений | 0 тем | 0 |

Клип с обеими темами всегда выигрывает у клипа с одной темой, независимо от количества тегов.

## Структура файлов

```
index.ts              Оркестратор, обработчики сигналов, глобальная очистка
src/audio.ts          Анализ темпа, транзиентов (пиков) и подбитов
src/indexer.ts        Deep Vision анализ, GPS/SRT парсинг, геокодирование
src/director.ts       Анализ промпта, topics, AI-директивы, планирование
src/color_grading.ts  Histogram matching CDF, temporal smoothing, LUT curves
src/render.ts         FFmpeg пайплайн, Chunked Assembly, Stable Zoom
src/ui/               Интерфейс на OpenTUI (React 19 + Bun FFI)
result/               Логи завершённых и упавших задач (<timestamp>_done.log)
```

## Кэширование и очистка

- **Метаданные**: Сайдкары `.json`. Версия кэша 7 (включает адреса и GPS).
- **Логи**: При завершении или ошибке сохраняются в `result/` с таймстемпом, промптом и длительностью каждого шага.
- **Терминал**: При выходе выполняется `renderer.destroy()` и Hard Reset (`\x1bc`).
- **Ошибки**: Экран `FATAL ERROR` удерживается до нажатия клавиши.
