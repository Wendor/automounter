# Automounter

AI-монтажёр видео: анализирует дроновые съёмки, понимает запрос на естественном языке и автоматически собирает музыкальный клип с синхронизацией по ритму.

---

## Требования

- **Bun** (обязательно для OpenTUI FFI):
  ```bash
  curl -fsSL https://bun.sh/install | bash
  ```
- **Ollama** с установленной vision-моделью:
  ```bash
  ollama pull llava:13b
  ollama serve
  ```
- macOS с Apple Silicon рекомендуется (аппаратное кодирование видео). Linux/Windows использует `libx264`.

## Установка

```bash
bun install
```

## Использование

### Интерактивный режим (OpenTUI)

```bash
bun start
```

Открывается современный TUI-интерфейс в духе OpenCode:
- **Project Config**: Текущие настройки (пути, промпт, качество).
- **Activity Log**: Детальный лог работы с автоматическим замером времени каждого шага и поддержкой скролла.
- **Pipeline**: Сайдбар со статусом каждой стадии и динамическим выводом результатов (BPM, количество лиц, локация).
- **Progress**: Адаптивный прогресс-бар процесса.

### Пакетный / скриптовый режим

```bash
bun start -- --batch
```

### Индексация без монтажа

```bash
# Индексировать новые файлы (Глубокий анализ + Геоданные)
bun start -- --index-only --input /path/to/vid

# Полная переиндексация (очистка кэша)
bun start -- --reindex --input /path/to/vid
```

## Логика эффектов и ритма

| Стиль/Энергия | Длина кадра   | Переходы / Эффекты             |
|---------------|---------------|--------------------------------|
| `epic` / High | 4–8 ударов    | flashIn, cut, dynamic zoom     |
| `dynamic`     | 4–8 ударов    | cut, zoomIn/Out (fast)         |
| `lyrical`     | 8–16 ударов   | dissolve, slow zoom, slow-mo   |
| `calm`        | 16+ ударов    | dissolve, steady shots         |

## Структура файлов

```
index.ts              Оркестратор, обработчики сигналов, глобальная очистка
src/audio.ts          Анализ темпа, транзиентов (пиков) и подбитов
src/indexer.ts        Deep Vision анализ, GPS/SRT парсинг, геокодирование
src/director.ts       AI-директивы фильтрации, творческое планирование
src/render.ts         FFmpeg пайплайн, Chunked Assembly, Stable Zoom
src/ui/               Интерфейс на OpenTUI (React 19 + Bun FFI)
```

## Кэширование и очистка

- **Метаданные**: Сайдкары `.json`. Версия кэша 7 (включает адреса и GPS).
- **Терминал**: При выходе выполняется автоматический `renderer.destroy()` и Hard Reset (`\x1bc`) для предотвращения мусора от скролла.
- **Ошибки**: Экран `FATAL ERROR` удерживается до нажатия клавиши для удобства чтения.
